# Powershell Analysis Lab Report

- Q1) What is the SHA256 hash value for the PowerShell script file?
  - Running sha256sum on terminal will give the sha256 hash for the file

- Q2) What email address is used to send and receive emails?
  - Using nano will reveal the email addresses used to send and receive emails

- Q3) What is the password for this email account?
  - Using nano will reveal the password for the email address

- Q4) What port is used for SMTP?
  - Using nano will reveal the port used for SMTP

- Q5) What DLL is imported to help record keystrokes?
  - Using nano will reveal the dll used to help record keystrokes

- Q6) What directory is the generated txt file put in?
  - Using nano will reveal the directory used to keep the generated .txt file
